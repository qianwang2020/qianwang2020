使用 Composer(PHP)

2020-03-08 2 min read # PHP # 学习

Composer 可以用来安装管理 PHP　项目中所需的各类依赖。

安装

首先需要先安装 PHP 的 CLI 程序，然后我们使用 PHP-CLI 来安装 Composer。 下载 Composer 的安装程序，实际上是一个 PHP 脚本:

php -r "copy('https://install.phpcomposer.com/installer', 'composer-setup.php');"

执行这个脚本文件:

php composer-setup.php

如果提示缺少相关 PHP 的扩展，自行安装, 至少它需要如下扩展:

sudo yum install php-mbstring php-json -y

然后将 composer.phar 移动到 Path 环境变量下的目录，以方便全局调用:

mv composer.phar /usr/bin/composer

最后使用 Composer 查看其版本，如果正常输出版本证明安装成功:

$ composer --version
Composer version 1.10.10 2020-08-03 11:35:19
# 删除 Composer 的安装文件
$ php -r "unlink('composer-setup.php');"

使用 Composer 安装框架

然后我们可以使用 Composer 来安装一个框架，体验其便捷之处，这里以 ThinkPHP 为例，其他的框架或项目可以再 Packagist.org 上查找:

composer create-project topthink/think think

如果安装进度非常慢，甚至是失败，可以尝试使用中国镜像:

composer config -g repo.packagist composer https://mirrors.aliyun.com/composer/

然后我们就完成了对 ThinkPHP 框架的安装。
