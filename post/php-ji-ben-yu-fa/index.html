PHP 基本语法

2020-09-08 2 min read

命名空间

命名空间的目的是为了防止类或者函数重名，PHP 采用的方法就是使用 namespace 关键字给类名加上前缀。

数组

示例和说明如下:

<?php
// 定义一个数组
// 注意： 数组在 PHP 中的地位非常高，它包括了在Python 中的元组、列表、字典等数据类型
$arr = [1, 2, 3];
// 你也可以看到这样的写法, 但已经很少有人这么写了
$arr = array(1, 2, 3);
// 可以通过 var_dump() 打印
var_dump($arr);
// 像 var_dump 这样的称谓函数，都是 PHP 定义的
// 在 PHP中有几千个这样的函数，非常之多，比如我们用的　sha1 加密函数
// 数组的形式多种多样，比如字典的形式
$arr = ['a' => 'b', 'c' => 'd'];
// 访问数组
var_dump($arr['a']);  // 输出 b
// 数组中可以嵌套数组
$arr = [
    'user' => [
        'name' => 'bob',
        'age' => 14
    ]
];
var_dump($arr['user']['name']);  // 输出 bob
// 有很多函数是为了数组服务的
// implode 是将数组转为字符串，使用 分隔符分割
echo implode('-', [1, 2, 3]);   // 输出 1-2-3
var_dump(explode('-', '1-2-3')); // 和 implode 相反，是将字符串通过指定的分隔符转为数组
// 在 PHP 中最方便的就是遍历数组
foreach ([1, 2, 3] as $num) {
    echo $num;    // 将会分别输出 1 2 3
}
// 数组的下标可以是数值索引，从0 开始，也可以是任意的字符串,称之为 key
$arr = [
  'name' => 'bob',
  'age' => 23
];
foreach ($arr as $key => $value) {
    echo $key . '/' . $value;   // 输出: name/bob 以及 age/23
}
