近期学习的Linux命令

2020-07-25 3 min read # 学习
  1. mkdir 创建文件夹
    mkdir aa
  2. touch 创建文件
    touch a.txt
  3. cd 切换目录
    cd home
    cd /
    cd ~
  4. cat 查看文件内容
    cat a.txt
  5. 登陆服务器
    ssh root@47.100.18.25
  6. tail 查看文件尾部内容
    tail -n1 /etc/passwd
  7. ls 显示指定工作目录下的内容及属性信息
    sudo ls -al /etc/skel
    -a 显示所有文件及目录 (包括以“.”开头的隐藏文件)
    -l 使用长格式列出文件及目录信息
  8. echo 输出字符串或提取Shell变量的值
    echo '' alias CD_TEST='cd ~/test' " >> ~/.bashrc
    追加内容 alias CD_TEST='cd ~/test' 到bashrc中
  9. alias 设置命令别名
    alias -p
    查看系统已经设置的别名
  10. su 切换用户
    su - test
    切换到 wq 用户，并改变 wq 用户的环境变量，- 表示切换用户的环境变量
  11. useradd 创建用户
    useradd -m test
  12. sudo passwd test
    设置用户密码
  13. userdel 删除用户
    sudo userdel -r test
    userdel 可删除etc/passwd 下的用户信息，但不会删除家目录
    -r 可以删除相关的目录信息
  14. 创建新用户并加入新创建的组
    sudo useradd test_user 创建新用户
    sudo groupadd test_group 创建新的用户组
    sudo usermod -G test_group test_user 将新用户加入新的组
    sudo groupmod -n test_mod_group test_group 将test_group 组名
    改为test_mod_group
  15. 查看文件权限
    ls -la test_file
  16. chmod 分配权限
    sudo chmod u-r test_file
    u -文件所属者
    g -文件的所属组
    o -其他用户
  17. 查看发行版的详细信息
    lsb_release -a
  18. 查看系统发行版本
    less /etc/issue q 退出
    cat /etc/issue
  19. 查看软件版本信息
    python --version
  20. 查看是否安装了某个软件包
    sudo rpm -qa python
  21. 卸载已经安装的软件包
    sudo rpm -e python --nodeps --nodeps 避免去检查相关的依赖环境导致不能卸载
  22. 安装软件
    sudo yum install python36 -y
  23. 服务器重装后keys改了终端连接不上的方法
    ssh-keygen -R 在本机执行此命令即可
