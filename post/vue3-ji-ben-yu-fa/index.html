Vue3 基本语法

2019-07-10 4 min read # JavaScript # 学习

Vue 第三个版本已经进入到了 Beta 版本，可以预期举例发布不远矣。这一篇文档将基于 Vue3 这个版本来讲述其基本的语法概念。

在正式开始之前，需要准备好开发环境，只需要有浏览器即可。另外，需要下载 Vue3 的 JS 文件，下载地址如下:

https://unpkg.com/vue@3.0.0-beta.17/dist/vue.global.js

另外，在学习的过程中，我们需要抓住一个关键点，类似于 Vue 这样的框架都是数据驱动的。什么是数据驱动，就是围绕着如何将后台的数据通过 Javascript 展现到页面上这个话题展开的。

Hello World

继承传统，我们还是从一个 Hello World 的示例开始讲，创建一个 Html 文件，然后引入我们的 Vue3 的 JS:

<!DOCTYPE html>
<html lang="en">
<head>
        <meta charset="UTF-8">
        <title>demo</title>
        <script src="vue.global.js"></script>
</head>
<body></body>
<script></script>
</html>

在 <body></body> 标签中编写页面结构:

  <div id="app">
    {{message}}
  </div>

最后，我们来编写脚本:

const message = "hello world"
const {createApp} = Vue
const app = {
    setup() {
        return {
            message
    }
  }
}
createApp(app).mount("#app")

然后我们去浏览这个页面，显示如下:

{{message}} ， 我们可以使用 {{}} 包裹一个变量名的形式来展示 JavaScript 中的数据。

在 JavaScript 部分，我们定义了数据 message , 然后通过定义一个 app 的变量，并在变量中定义一个入口函数 setup ,最后在入口函数中将返回一个对象，并将数据在对象中返回出去。

但是，我们一定要触发这个对象的在页面上的初始化，只有初始化之后，我们才可以用这个对象中定义的数据和方法，初始化的这个动作我们叫做挂载(mount)。如下代码:

createApp(app).mount('#app')

也就是说，只有 #app 这个节点可以使用我们的创建的 Vue 对象。

指令

什么是指令呢?就类似于 HTML 的标签中的属性，他们都实现了特定的功能。

v-text

首先我们来看一个名为 v-text 的指令，我们来改写一下上面 Hello World 的示例:

  <div id="app">
    /** 把原先的 {{message}} 替换成下面这行代码 **/
    <div v-text="message"></div>
  </div>

然后我们去看页面的话，会发现界面还是一样的，没有改变。 v-text 用来将变量替换指定标签的内容为变量的值。

v-html

和 v-text 的区别是，它可以展示 HTML 内容。加入我们原先的 message 变量是 <h1>Hello World</h1> ，使用 v-text 会原封不动的显示出来。而使用 v-html 可以以HTML 的形式展示出来:

const message = '<h1>Hello World</h1>'

然后修改一下 HTML, 将原先的 v-text 换成 v-html ：

 <div id="app">
   <div v-html="message"></div>
 </div>

页面显示如下:

    • Hello World
    • 指令
      • v-text
      • v-html
