使用 Gridea 搭建 GitHub Pages

2019-01-12 2 min read # 随笔 # 学习

Github 是全球最大的软件开源社区，我们可以在这个上面去管理我们的代码仓库、和来自世界各地的人共同编程、学习、交流。

而 Github Pages 是 Github 推出的一个功能，可以让我们使用 Github 的仓库为基础搭建个人主页，发表文章，供人评论。

Gride 是 Github 上的一个开源的项目，可以使用它来快速地创建 GitHub Pages，以及方便地书写文章并发布到主页上。

下载并安装 Gride

Gride 在 GitHub 上地仓库 getgridea/gridea 地 README 中提供了下载的途径，我们可以根据不同的操作系统下载各自的安装包进行安装。

具体地安装过程就不在此叙述了，和安装普通的软件也没有什么区别。
