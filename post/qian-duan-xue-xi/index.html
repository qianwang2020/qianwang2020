初学前端React

2020-08-26 2 min read

目前前端的发展很快很快，相比之下，后端技术的发展很慢，各种工具技术，对于初学是由难度的。
和后端不一样，前端可以选择的框架不多，后端的话，光 PHP 就有好多框架。前端框架国内常用的有两个，分别是阿里的 Vue 和 国外脸书的 React，这次我们选择的是 React。umi 以及 后面安装的 antd 都是阿里出的，基于 React 框架的工具库。

cd ~
mkdir antd
yarn create umi
预览一下效果
安装一直很慢的情况下，你也可以 Ctrl + C中止，执行下面这条
yarn config set registry 'https://registry.npm.taobao.org'命令，然后重新执行yarn create umi命令。

在安装的过程中，会让你选，直接回车就好。然后要选择版本，我们选择 V4

选择使用的语言，我们选择 TypeScript
然后选择需要的模块，我们选择 complete 就是全部的意思，然后就把这个后台项目拷贝下来了，然后你会看到有这些文件

然后使用 yarn install 来安装依赖，这可能需要一些时间。在安装之前，先执行yum install make -y需要说明的是，前端的发展很快，很快指的是发展出一系列的工具链，没有人能完全掌握这些工具链，所以只要会使用这些工具就好了。
